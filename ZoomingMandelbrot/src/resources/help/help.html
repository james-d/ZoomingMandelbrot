Mandelbrot Set Application

This application allows you to explore the Mandelbrot Set.

Functionality

  • User Controls

  • Iteration Level
  • Julia Sets

User Controls

Most functionality is available from either a context menu (which you can bring up by right-clicking on the application) or via keyboard shortcuts (using the Control key on a computer running Windows, or Command on an Apple Mac). Zooming functionality is available by double-clicking the Mandelbrot image. The sections below describe the functionality and the context menu items or keyboard shortcuts used to invoke it.

Zooming

(Top)
Functionality Controls Notes
Zoom in by a factor of 4 Double-click on the zoom point Will zoom out if "Reverse Zoom" is enabled
Zoom out by a factor of 4 Shift-double-click on the zoom point Will zoom in if "Reverse Zoom" is enabled
Reverse zoom Select from context menu (right-click) or Ctrl-Z Reverses double-click and shift-double-click
Reset Select from the context menu (right-click) or Ctrl-R Resets to the original image

Iteration Level

(Top)
Functionality Controls Notes
Guess iteration level automatically Check the "Guess" check box next to the iteration field in the control panel Makes a best-effort attempt to find a good iteration level, based on the size of the viewing portion only.
Manually enter iteration level With the "Guess" checkbox unchecked, enter the iteration level in the text field. A minimum level of 10 is enforced. The iteration level takes effect on the next computation of the Mandelbrot set.

Julia Sets

(Top)
Functionality Controls Notes
Toggle tracking of Julia Set Select from the context menu (right-click) or Ctrl-J Useful for "freezing" the Julia Set display, for example prior to exporting the image

Image Export

(Top)
Functionality Controls Notes
Export Mandelbrot Set image From the context menu (right-click menu) or by Ctrl-S Saves a png format image of the current Mandelbrot Set
Export Julia Set image From the context menu (right-click menu) or by Ctrl-Shift-S Saves a png format image of the current Julia Set

Control Panel

(Top)
Display Description
Size Shows the size (width, which is equal to the height) of the current Mandelbrot set image
Frames per second During zooming, shows (approximately) the number of frames being generated every second.
Progress bar. During zooming, shows the progress of the zoom

Summary of keyboard shortcuts

(Top)

These shortcuts are listed as Ctrl-Key. On an Apple Mac, use Command-Key.

Shortcut Action
Ctrl-J Toggle tracking Julia Set
Ctrl-Z Toggle Zoom In/Out
Ctrl-R Reset to original image
Ctrl-S Save Mandelbrot Image
Shift-Ctrl-S Save Julia Image
Ctrl-H Show this help page
Ctrl-Q Exit the application
